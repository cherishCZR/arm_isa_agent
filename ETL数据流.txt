数据模型关系:

  ┌─────────────┐
  │ instructions │  (1 条 ARM 指令)
  │ ─────────── │
  │ id (PK)     │
  │ name        │──── 1:N ────┐
  │ mnemonic    │             │
  │ title       │      ┌──────▼──────┐
  │ instr_class │      │  encodings  │ (每条指令可能有多种编码)
  │ is_alias    │      │ ─────────── │
  │ aliasto_id ─┼─┐    │ id (PK)     │
  └─────────────┘ │    │ instr_id(FK)│──── 1:N ────┐
                  │    │ name        │             │
                  │    │ bit_pattern │      ┌──────▼──────┐
                  │    │ bitfields   │      │  operands   │ (每个编码有多个操作数)
                  │    │ asm_template│      │ ─────────── │
                  │    │ width       │      │ id (PK)     │
                  │    └────────────┘      │ encoding_id  │
                  │                        │ symbol      │
                  │  M2M                   │ type        │
                  │  ┌──────────────────┐  │ width       │
                  │  │ instr_features   │  │ description │
                  │  │ ─────────────── │  │ values      │
                  │  │ instruction_id   │  └─────────────┘
                  │  │ feature_name ────┼──▶ features (架构特性)
                  │  │ level            │
                  │  └──────────────────┘
                  │
                  │  1:N
                  ├──────▶ constraints    (UNPREDICTABLE 约束)
                  │        ├── id (PK)
                  │        ├── instruction_id (FK)
                  │        ├── constraint_type
                  │        └── expression (JSON)
                  │
                  │  1:N
                  └──────▶ pseudocode     (Decode/Execute 伪代码)
                           ├── id (PK)
                           ├── instruction_id (FK)
                           ├── section_type
                           └── pseudocode_text

  ChromaDB:
  ┌──────────────────────┐
  │ instruction_cards    │  (每条指令的 Markdown 文档 + 向量)
  │ ────────────────────│
  │ id                   │
  │ instruction_id (FK   │
  │   → instructions.id) │
  │ document (Markdown)  │
  │ embedding ([1024d])  │
  │ metadata (JSON):     │
  │   ├── mnemonic       │
  │   ├── instr_class    │
  │   ├── architecture   │
  │   ├── features       │
  │   └── ...            │
  └──────────────────────┘
