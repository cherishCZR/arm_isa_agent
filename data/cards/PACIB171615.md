## PACIB171615
_ARM A64 Instruction_

**Title**: PACIB171615 -- A64 | **Class**: `general` | **XML ID**: `PACIB171615`

**Architecture**: `FEAT_PAuth_LR` (ARMv9.5)

**Summary**: Pointer Authentication Code for instruction address, using key B

**Description**:
This instruction computes and inserts a pointer authentication code for
an instruction address, using two modifiers and key B. The address is in X17.
The 64-bit value of modifier1 is the value in X16. The 64-bit value
of modifier2 is the value in X15.
