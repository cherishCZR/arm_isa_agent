关键数据流:
═══════════════════════════════════════════════════════════════════════════════════════════

  对话模式 (Agent Chat):
  ──────────────────────
  User → CLI/API → LangGraph Agent → tools/ → kb/ → Summarizer → Response

  Compiler Verification 模式 (一键验证):
  ──────────────────────────────────────────
  User → POST /api/generate_testcase → VerificationOrchestrator
       → Stage1(rag/) → Stage2(planning/analyzer) → Stage3(planning/planner)
       → Stage4(generation/generators) → Stage5(review_generation/reviewer)
       → Stage6(repair_loop) → Stage7(report) → VerificationReport JSON
